## 📖 Manual Commands Cheatsheet

### Subdomain Enumeration

```bash
subfinder -d target.com -silent -all -o subs.txt

import json,sys
for e in json.load(sys.stdin):
  for n in e.get('name_value','').split('\n'):
    n=n.strip().lstrip('*.').lower()
    if n.endswith('target.com'): print(n)
" | sort -u
```

### Port Scanning

```bash
nmap -sS -T4 --open --top-ports 1000 target.com
nmap -sS -T3 --open -p- target.com                    # all ports
nmap -sV -sC --script=vuln --open -p 80,443 target.com  # vuln scripts
sudo masscan target.com -p 1-65535 --rate 10000         # fastest
sudo nmap -sU --top-ports 50 target.com                 # UDP
```

### Live Host Probing


## 🔧 Troubleshooting

### Permission Denied

```bash
chmod +x ~/autopwn/hackerofhell_ultra.sh
sudo bash ~/autopwn/hackerofhell_ultra.sh -t target.com
```

### Go Tools Not Found After Install

```bash
echo 'export PATH=$PATH:$HOME/go/bin' >> ~/.bashrc
source ~/.bashrc
ls ~/go/bin/
which subfinder httpx nuclei dalfox
```

### Nuclei Templates Missing

```bash
nuclei -update-templates
find ~/nuclei-templates -name "*.yaml" | wc -l  # should be 7000+
```

### Report Not Generated

```bash
# Check if scan completed Phase 20
tail -50 ~/hackerofhell_ultra/target.com/hackerofhell_ultra.log
# Force regenerate report
python3 ~/hackerofhell_ultra/target.com/findings_ultra.json
# Find all reports
find ~/hackerofhell_ultra -name "HACKEROFHELL_*.html"
```

### Tool Install Fails (Go Version Too Old)

```bash
go version  # need 1.19+
# Install latest:
wget -q https://go.dev/dl/go1.22.5.linux-amd64.tar.gz
sudo rm -rf /usr/local/go
sudo tar -C /usr/local -xzf go1.22.5.linux-amd64.tar.gz
echo 'export PATH=$PATH:/usr/local/go/bin' >> ~/.bashrc
source ~/.bashrc
go version
```

### sqlmap Too Slow

```bash
# Skip sqlmap with --skip-heavy
sudo bash hackerofhell_ultra.sh -t target.com --skip-heavy
# Or reduce depth
# Edit Phase 07 in script: change --level=2 --risk=2 to --level=1 --risk=1
```

### Scan Taking Too Long

```bash
# Use --skip-heavy (removes sqlmap + brute force)
sudo bash hackerofhell_ultra.sh -t target.com --skip-heavy

# Reduce threads
sudo bash hackerofhell_ultra.sh -t target.com -T 10

# Skip deep port scan
# Remove --deep flag
sudo bash hackerofhell_ultra.sh -t target.com  # without --deep
```

### Target Behind Cloudflare

```bash
# Check origin via subdomain IPs
for sub in ftp mail smtp origin direct api admin staging; do
  ip=$(dig +short $sub.target.com | grep -E '^[0-9]+\.' | head -1)
  [[ -n "$ip" ]] && echo "$sub.target.com → $ip"
done

curl "https://securitytrails.com/domain/target.com/history/a" 2>/dev/null
```
